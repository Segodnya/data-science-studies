→ задача проекта:
примерив на себя роль аналитика в компании кинопроката, провести свой первый анализ данных с помощью pandas.

27 из 27 (100%)

В ходе выполнения заданий приноровился работать со строковыми данными, преобразовывать их и формировать новые объекты.
В ряде задач для решения я прибегнул к сводными таблицам. Пролистывал Slack - такого варианта решения не встречал.
